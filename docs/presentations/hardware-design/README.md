# AudioLab Hardware Design Presentation

Open `index.html` in a browser to present the deck.

The deck is self-contained and uses local images generated from:

- `hardware/docs/schematics.pdf`
- `hardware/docs/pcb.pdf`
- `assets/AudioLab-above.png`
- `assets/AudioLab-below.png`

Use the left and right arrow keys, `PageUp` / `PageDown`, or the on-screen
buttons to move between slides. Use browser print-to-PDF if a PDF export is
needed.
